`virtual-cursor` smooths cursor movement at mark boundaries, especially in rich text editors that combine inline formatting with custom node views.

## Install

```bash
pnpm add @citolab/prose-extensions
```

## Usage

```ts
import { createVirtualCursor } from '@citolab/prose-extensions/virtual-cursor';

const virtualCursorPlugin = createVirtualCursor();
```

Add the returned ProseMirror plugin to your plugin list. If you use ProseKit, wrap it with `definePlugin()` in your local extension assembly.

## When to use

Use this when the caret gets stuck or behaves awkwardly around mark transitions such as bold, italic, or other inline formatting boundaries.
