`qti-interaction-insert-menu` is a dropdown button for inserting supported QTI interaction nodes into the editor.

## Install

```bash
npx shadcn@latest add https://qti-editor.citolab.nl/r/qti-interaction-insert-menu.json
```

The registry command installs the component source and its package dependencies into your app.

## Usage

```ts
import '@/components/interaction-insert-menu';

const menu = document.querySelector('qti-interaction-insert-menu');
menu.editor = editor;
```
