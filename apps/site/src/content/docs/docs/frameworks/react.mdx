This guide shows a minimal React integration pattern for QTI Editor packages.

## 1. Install dependencies

```bash
pnpm add @qti-editor/core @qti-editor/ui @qti-editor/prosekit-integration lit prosekit
```

## 2. Create `QtiEditorHost.tsx`

```tsx
import { useEffect, useRef } from 'react';
import { createEditor } from 'prosekit/core';

export function QtiEditorHost() {
  const mountRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    if (!mountRef.current) {
      return;
    }

    const editor = createEditor({
      extension: [],
      defaultContent: '<p>Start authoring QTI content.</p>',
    });

    mountRef.current.appendChild(editor.mount());

    return () => {
      editor.destroy();
    };
  }, []);

  return <div ref={mountRef} />;
}
```

## 3. Place the host in your app (`App.tsx`)

```tsx
import { QtiEditorHost } from './QtiEditorHost';

export default function App() {
  return (
    <main>
      <h1>QTI Authoring</h1>
      <QtiEditorHost />
    </main>
  );
}
```

## Notes

- Keep editor configuration in a dedicated module when you add custom interactions.
- Add toolbar and side panel wiring after base rendering is stable.
